This project takes data from the careers of Lebron James, Kobe Bryant, and Michael Jordan. As these three players are regarded as some of the greatest to play the game, it seemed fair to compare some of their statistics.

The link to the dataset:
https://www.kaggle.com/datasets/xvivancos/michael-jordan-kobe-bryant-and-lebron-james-stats?resource=download&select=advanced_stats.csv
